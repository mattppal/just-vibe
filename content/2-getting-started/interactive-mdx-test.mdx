# Interactive MDX Example

This page demonstrates the usage of interactive React components directly within MDX documents.

## Interactive Tabs

Use tabs to organize related content in a space-efficient way:

<Tabs>
  <TabsList>
    <TabTrigger value="overview">Overview</TabTrigger>
    <TabTrigger value="example">Example</TabTrigger>
    <TabTrigger value="api">API Reference</TabTrigger>
  </TabsList>
  
  <TabItem value="overview" title="Overview">
    The overview provides a high-level introduction to the concept.
    
    - Key benefit one
    - Key benefit two
    - Key benefit three
  </TabItem>
  
  <TabItem value="example" title="Example">
    ```javascript
    // Example code
    function greeting(name) {
      return `Hello, ${name}!`;
    }
    
    greeting('World');
    ```
  </TabItem>
  
  <TabItem value="api" title="API Reference">
    | Parameter | Type | Description |
    |-----------|------|-------------|
    | name | string | The name to greet |
    | options | object | Optional configuration |
  </TabItem>
</Tabs>

## Code Blocks with Syntax Highlighting

Code blocks support syntax highlighting:

```typescript
interface User {
  id: number;
  name: string;
  email: string;
  roles: string[];
}

function getUserInfo(userId: number): Promise<User> {
  return fetch(`/api/users/${userId}`)
    .then(response => {
      if (!response.ok) {
        throw new Error('Failed to fetch user');
      }
      return response.json();
    })
    .then(data => data as User);
}
```
