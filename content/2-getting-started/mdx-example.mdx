## What is MDX?

MDX is a format that combines Markdown with JSX (JavaScript XML) syntax. It allows you to use React components directly in your Markdown content.

### TypeScript

```typescript
interface User {
  id: number;
  name: string;
  email: string;
  role: 'admin' | 'user' | 'guest';
}

class UserService {
  private users: User[] = [];
  
  constructor() {
    // Initialize service
  }
  
  public addUser(user: User): void {
    this.users.push(user);
  }
  
  public getUserById(id: number): User | undefined {
    return this.users.find(user => user.id === id);
  }
}
```

### React Component

```jsx
import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  
  return (
    <div className="counter">
      <h2>Counter: {count}</h2>
      <button onClick={() => setCount(count + 1)}>Increment</button>
      <button onClick={() => setCount(count - 1)}>Decrement</button>
    </div>
  );
}

export default Counter;
```

### Python

```python
class Animal:
    def __init__(self, name, species):
        self.name = name
        self.species = species
        
    def make_sound(self):
        pass
    
    def __str__(self):
        return f"{self.name} is a {self.species}"

class Dog(Animal):
    def __init__(self, name, breed):
        super().__init__(name, species="Dog")
        self.breed = breed
        
    def make_sound(self):
        return "Woof!"

# Create a new dog
my_dog = Dog("Rex", "German Shepherd")
print(my_dog)  # Output: Rex is a Dog
print(my_dog.make_sound())  # Output: Woof!
```

## Importing Components

You can import and use React components in your MDX files:

```jsx
import { Button } from '@/components/ui/button';

<Button variant="primary">Click me!</Button>
```

## Line Highlighting

You can highlight specific lines in code blocks with the line highlighting syntax:

```javascript {2-4,7}
function process() {
  // This line is highlighted
  // This line is highlighted too
  const result = performCalculation();
  
  if (result > 100) {
    // This line is also highlighted
    sendAlert('High value detected');
  }
  
  return result;
}
```
